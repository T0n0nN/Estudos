## Como ver no seu PC

Opção A (mais simples):

- No VS Code, clique com botão direito no `index.html` → **Open with Live Server** (se tiver a extensão).

Opção B (sem extensão):

- Abra um terminal na pasta e rode:
  - `python -m http.server 5500`
- Depois abra: `http://localhost:5500`

## Hospedagem grátis (recomendado)

### Opção 1 — GitHub Pages (grátis)

Repositório usado:

- `https://github.com/T0n0nN/rafael_tonon`

Publicando (modo simples):

1. Garanta que `index.html` e `styles.css` estão na **raiz** do repositório.
2. Faça commit e push para a branch `main`.
3. No GitHub: **Settings** → **Pages**

   Source: `Deploy from a branch`

   Branch: `main` / folder: `/ (root)`

4. Aguarde alguns minutos e abra a URL do site.

URL esperada:

- `https://t0n0nn.github.io/rafael_tonon/`

Observação:

- Para ficar 100% grátis, o repositório normalmente precisa estar **público**.

## Como atualizar o site no GitHub Pages

Sempre que você alterar o `index.html` e/ou `styles.css`, faça:

1. Abra o terminal **na pasta do repositório clonado** (`rafael_tonon`).
2. Rode os comandos.

### Como abrir o terminal na pasta do repositório (Windows)

Opção A — Pelo VS Code (mais fácil):

1. Abra o VS Code.
2. Clique em **File** → **Open Folder...**
3. Selecione a pasta do repositório clonado (ex.: `rafael_tonon`) e abra.
4. Clique em **Terminal** → **New Terminal**.

O terminal integrado do VS Code já abre apontando para a pasta do repositório.

Opção B — Pelo Explorador de Arquivos:

1. Abra a pasta `rafael_tonon` no Explorador.
2. Clique na barra de endereço (onde aparece o caminho da pasta).
3. Digite `powershell` e pressione Enter.

Isso abre um PowerShell já dentro da pasta.

Opção C — Pelo PowerShell usando `cd`:

1. Abra o PowerShell.
2. Vá até a pasta onde você clonou o repo.

Exemplo:

- `cd "C:\Users\SEU_USUARIO\Documents\rafael_tonon"`

- `git status`
- `git add .`
- `git commit -m "Atualizar site"`
- `git push`

Depois disso, o GitHub Pages atualiza sozinho (normalmente em 1–5 minutos).

### Opção 2 — Netlify (grátis)

1. Acesse Netlify e crie uma conta.
2. "Add new site" → "Deploy manually".
3. Arraste a pasta (ou os arquivos) para publicar.
